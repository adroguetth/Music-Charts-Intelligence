## 📋 Descripción General

Este proyecto es el cuarto componente del sistema de inteligencia de charts de YouTube. Toma la base de datos semanal enriquecida (del Script 3) y **genera automáticamente notebooks profesionales de Jupyter** con análisis visual completo e insights generados por IA tanto en inglés como en español.

El sistema produce dos notebooks completamente ejecutados por semana que contienen **más de 25 visualizaciones**, **12 secciones de análisis** y **comentarios generados por IA** mediante la API de DeepSeek, todo almacenado en caché para evitar llamadas redundantes a la API.

### Características Principales

- **📊 Más de 25 Visualizaciones**: Gráficos estáticos usando matplotlib, seaborn y squarify (sin necesidad de JavaScript)
- **🤖 Insights Generados por IA**: La API de DeepSeek proporciona comentarios inteligentes para cada sección de análisis
- **🌐 Salida Bilingüe**: Genera notebooks en inglés y español desde una sola ejecución
- **💾 Caché Inteligente**: Sistema de caché por semana e idioma evita llamadas redundantes a la API
- **📅 Automatización Semanal**: Completamente automatizado vía GitHub Actions, se ejecuta cada lunes
- **🗂️ Retención de Ventana Deslizante**: Mantiene solo los últimos 6 notebooks por idioma automáticamente
- **📈 Resumen Ejecutivo**: Resumen detallado de 30 líneas con recomendaciones accionables
- **🎨 Estilo YouTube**: Las visualizaciones usan la paleta de colores de YouTube (#FF0000, #F9F9F9, etc.)
- **🔧 Cero JavaScript**: Todos los gráficos son estáticos (matplotlib/seaborn), compatibles con vista previa de GitHub
- **💾 Agnóstico de Base de Datos**: Funciona con cualquier base de datos enriquecida del Script 3

### **Diagrama 1: Vista General del Flujo Principal**

<img src="https://raw.githubusercontent.com/adroguetth/Music-Charts-Intelligence/main/Documentation_ES/Diagramas/4_1.weekly_charts_notebook_generator/EN-Diagram%201.png" alt="Vista General del Flujo Principal" width="600">

Este diagrama muestra el **pipeline de alto nivel** de todo el sistema:

1. **Entrada**: Lee la base de datos semanal enriquecida (`youtube_charts_YYYY-WXX_enriched.db`)
2. **Carga de Datos**: Carga datos, calcula engagement, extrae características temporales
3. **Resúmenes de Datos**: Precalcula métricas para prompts de IA (top países, engagement por género, etc.)
4. **Verificación de Caché**: Para cada sección, verifica si el insight existe en caché
   - **Si está en caché** → Usa insight almacenado ✅
   - **Si no está en caché** → Consulta API de DeepSeek → Guarda en caché
5. **Generación de Notebook**: Crea notebook Jupyter con celdas de código e insights de IA
6. **Ejecución de Notebook**: Ejecuta `nbconvert --execute` para incrustar todas las salidas
7. **Limpieza**: Elimina notebooks antiguos (mantiene últimos 6 por idioma)
8. **Commit a GitHub**: Automáticamente commitea y sube los cambios

### **Diagrama 2: Sistema de Caché de Insights de IA**

<img src="https://raw.githubusercontent.com/adroguetth/Music-Charts-Intelligence/main/Documentation_ES/Diagramas/4_1.weekly_charts_notebook_generator/EN-Diagram%202.png" alt="Sistema de Caché de Insights de IA" width="600">

Este diagrama detalla el **mecanismo de caché inteligente**:

1. **Solicitud de Sección**: El sistema necesita un insight para una sección específica (ej., "genre_engagement")
2. **Huella Digital de Datos**: Genera hash MD5 del subconjunto de datos relevante:
   - `general_stats` → hash de (total_canciones, total_vistas, total_likes)
   - `top_countries` → hash del string de top 10 países
   - `genre_engagement` → hash de tasas de engagement por género
3. **Búsqueda en Caché**: Verifica archivo JSON con clave `{sección}_{hash}`
   - **Acierto de caché** → Retorna insight inmediatamente (sin llamada API)
   - **Fallo de caché** → Procede a API de DeepSeek
4. **Consulta a DeepSeek**: Envía prompt específico por idioma con resumen de datos
5. **Almacenamiento de Insight**: Guarda resultado con clave hash para ejecuciones futuras
6. **Aislamiento de Caché**: Archivos de caché separados por semana e idioma
   - `youtube_charts_2026-W14_en.json`
   - `youtube_charts_2026-W14_es.json`
