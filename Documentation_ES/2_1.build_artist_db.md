## 📋 Descripción General

Este script es el **segundo componente** del sistema de inteligencia de Charts de YouTube. Toma los nombres de artistas en bruto extraídos por el Script 1 y los **enriquece con metadatos geográficos y de género** consultando múltiples bases de conocimiento abiertas. El resultado es una base de datos estructurada y acumulativa de artistas con su país de origen y género musical principal.

### ¿Por qué una Arquitectura Modular?

El script original [`2_1.build_artist_db.py`](https://github.com/adroguetth/Music-Charts-Intelligence/blob/main/scripts/scripts_backup/2_1.build_artist_db_original.py) creció a **más de 10,000 líneas**, lo que lo hacía:

- **Difícil de mantener** (un solo cambio podía romper múltiples características)
- **Difícil de probar** (sin aislamiento entre componentes)
- **Lento de depurar** (los problemas requerían rastrear funciones masivas)

La versión refactorizada (`2_1.build_artist_db.py`) divide el código en **30+ módulos enfocados**, cada uno con una sola responsabilidad:

| Problema                                     | Solución                                    |
| :------------------------------------------- | :------------------------------------------ |
| Script monolítico de 10,000 líneas           | 30+ módulos pequeños (<300 líneas cada uno) |
| Lógica de API mezclada con lógica de negocio | Paquete `apis/` separado                    |
| Datos de país/género codificados             | Paquete `dictionaries/` externo             |
| Código duplicado entre funciones             | Paquete `utils/` compartido                 |
| Difícil añadir nuevas fuentes de datos       | Módulos API plug-and-play                   |

### Características Principales

- **Arquitectura Modular**: 30+ módulos enfocados en lugar de un script monolítico de 10,000 líneas
- **Búsqueda Multi-Fuente**: Consultas en cascada inteligentes a MusicBrainz, Wikipedia, Wikidata
- **Fallback con IA DeepSeek**: Usa la API de DeepSeek como último recurso (económico, ~$0.002 por 100 artistas)
- **Variaciones Inteligentes de Nombre**: Genera hasta 15 variaciones por artista para máxima tasa de coincidencia
- **Inteligencia Geográfica**: Detección de país desde 30,000+ términos (ciudades, gentilicios, regiones)
- **Clasificación de Géneros**: 200+ macro-géneros y 5,000+ mapeos de subgéneros con votación ponderada
- **Reglas Específicas por País**: Manejo especial para 50+ países (K-Pop para Corea del Sur, etc.)
- **Detección de Escritura**: Detección automática de idioma para escrituras no latinas
- **Actualizaciones Inteligentes**: Solo completa datos faltantes, nunca sobrescribe información correcta
- **Caché en Memoria**: Evita llamadas API redundantes dentro de una sesión
- **Optimizado para CI/CD**: Configurado específicamente para GitHub Actions

```python
#!/usr/bin/env python3
"""
Sistema de Detección de País + Género de Artistas - Orquestador Principal
"""

import sys
from pathlib import Path

# Agregar directorio padre a sys.path para importaciones de módulos
SCRIPT_DIR = Path(__file__).parent.absolute()
PROJECT_ROOT = SCRIPT_DIR.parent
if str(PROJECT_ROOT) not in sys.path:
    sys.path.insert(0, str(PROJECT_ROOT))

# Importaciones del paquete modular
from build_artist_db.config import CHARTS_DB_DIR, ARTIST_DB_PATH, setup_logging
from build_artist_db.utils.db_utils import (
    create_database, artist_in_database, insert_artist,
    count_artists_in_database, get_latest_chart_database, get_artists_from_chart_db,
)
from build_artist_db.country_detector import search_country
from build_artist_db.genre_detector import search_artist_genre

logger = setup_logging()


def main() -> None:
    """Función principal de ejecución."""
    logger.info("🎵 Sistema de Detección de País + Género de Artistas - MODO MODULAR")
    
    # 1. Inicializar base de datos
    create_database()
    
    # 2. Encontrar la base de datos de charts más reciente
    chart_db = get_latest_chart_database()
    if not chart_db:
        logger.error("❌ No se pudo obtener la base de datos de charts")
        sys.exit(1)
    
    # 3. Extraer artistas únicos
    chart_artists = get_artists_from_chart_db(chart_db)
    
    # 4. Procesar cada artista
    for artist in sorted(chart_artists):
        exists, db_country, db_genre = artist_in_database(artist)
        
        if exists and db_country and db_genre:
            logger.info(f"  ✅ {artist} → {db_country} | {db_genre}")
            continue
        
        # Buscar información faltante
        country, country_source = search_country(artist)
        genre, genre_source = search_artist_genre(artist, country)
        
        # Actualizar base de datos
        insert_artist(artist, country or "Unknown", genre, f"{country_source} | {genre_source}")
    
    # 5. Resumen final
    logger.info(f"📈 Total de artistas en base de datos: {count_artists_in_database()}")


if __name__ == "__main__":
    main()
```

- **Sin procesamiento de datos** (delegado a `utils/`)

#### 3.1 Utilidades de Texto (`utils/text_utils.py`)

**Propósito**: Normalización de cadenas, generación de variaciones de nombre y detección de escritura.

```python
"""
Utilidades de normalización de texto y generación de variaciones de nombre.
"""

import re
import unicodedata
from typing import List, Optional


def normalize_text(text: str) -> str:
    """Minúsculas, colapsa espacios en blanco, elimina puntuación."""
    if not text:
        return ""
    text = text.lower().strip()
    text = re.sub(r'\s+', ' ', text)
    text = text.strip('.,;:¿?¡!()[]"\'«»…–—')
    return text


def generate_name_variations(name: str) -> List[str]:
    """Genera variaciones básicas: original, sin acentos, sin puntos, sin guiones."""
    variations = [name]
    
    # Eliminar acentos
    no_accents = ''.join(c for c in unicodedata.normalize('NFD', name) 
                         if unicodedata.category(c) != 'Mn')
    if no_accents != name:
        variations.append(no_accents)
    
    # Eliminar puntos
    no_dots = name.replace('.', '')
    if no_dots != name:
        variations.append(no_dots)
    
    # Reemplazar guiones con espacios
    no_hyphens = name.replace('-', ' ')
    if no_hyphens != name:
        variations.append(no_hyphens)
    
    return list(dict.fromkeys(variations))


ARTIST_PREFIXES = {'dj': ['DJ', 'Dj', 'dj'], 'mc': ['MC', 'Mc', 'mc'], 
                   'lil': ['Lil', 'lil', 'LIL'], 'the': ['The', 'the', 'THE'], ...}


def remove_artist_prefixes(name: str) -> List[str]:
    """Elimina prefijos comunes (DJ, MC, Lil, The, etc.)."""
    variations = [name]
    for prefix_base, variants in ARTIST_PREFIXES.items():
        for variant in variants:
            if re.match(r'^' + re.escape(variant) + r'\s+', name):
                without_prefix = re.sub(r'^' + re.escape(variant) + r'\s+', '', name)
                if without_prefix:
                    variations.append(without_prefix)
    return list(dict.fromkeys(variations))


def generate_all_variations(name: str) -> List[str]:
    """Genera hasta 15 variaciones combinando todas las estrategias."""
    all_vars = set()
    for var in generate_name_variations(name):
        all_vars.add(var)
        for no_prefix in remove_artist_prefixes(var):
            all_vars.add(no_prefix)
            for var2 in generate_name_variations(no_prefix):
                all_vars.add(var2)
    
    result = [name]
    for var in sorted(all_vars - {name}, key=len, reverse=True):
        if var and len(var) > 1:
            result.append(var)
    return result[:15]


def detect_script_from_name(name: str) -> Optional[str]:
    """Detecta el sistema de escritura y devuelve el código de idioma ISO 639-1."""
    # Devanagari (Hindi, Marathi, Nepalí)
    if re.search(r'[\u0900-\u097F]', name):
        return 'hi'
    # Árabe / Urdu
    elif re.search(r'[\u0600-\u06FF]', name):
        if re.search(r'[گچپژ]', name):
            return 'ur'  # Caracteres específicos de Urdu
        return 'ar'
    # Cirílico (Ruso, Ucraniano, Búlgaro, Serbio)
    elif re.search(r'[\u0400-\u04FF]', name):
        if re.search(r'[іїєґ]', name):
            return 'uk'
        if re.search(r'ъ', name):
            return 'bg'
        if re.search(r'[ћђџљњ]', name):
            return 'sr'
        return 'ru'
    # Japonés (Hiragana/Katakana)
    elif re.search(r'[\u3040-\u309F\u30A0-\u30FF]', name):
        return 'ja'
    # Chino (Han) - distinguir de Coreano
    elif re.search(r'[\u4E00-\u9FFF]', name):
        if re.search(r'[\uAC00-\uD7AF]', name):
            return 'ko'
        return 'zh'
    # Hangul (Coreano)
    elif re.search(r'[\uAC00-\uD7AF]', name):
        return 'ko'
    # Latino con acentos (Español/Portugués)
    elif re.search(r'[áéíóúâêôàèìòùäëïöü]', name):
        if re.search(r'[çãõ]', name):
            return 'pt'
        return 'es'
    
    return None
```

**Por qué existe este módulo:**

- **Centraliza el procesamiento de texto** (usado por country_detector, genre_detector y las APIs)

- **Variaciones de nombre** mejoran las tasas de coincidencia entre diferentes fuentes de datos

- **Detección de escritura** guía la selección de idioma de Wikipedia (artistas coreanos → Wikipedia coreana)

  

#### 3.2 Utilidades de Base de Datos (`utils/db_utils.py`)

**Propósito**: Todas las operaciones SQLite en un solo lugar.

```python
"""
Operaciones de base de datos para la base de datos de artistas.
"""

import sqlite3
import re
from pathlib import Path
from typing import Optional, Tuple, Set

from ..config import ARTIST_DB_PATH, CHARTS_DB_DIR, logger
from .artist_parser import split_artists


def create_database() -> None:
    """Crea la tabla de artistas si no existe."""
    conn = sqlite3.connect(str(ARTIST_DB_PATH))
    cursor = conn.cursor()
    cursor.execute('''
        CREATE TABLE IF NOT EXISTS artist (
            name TEXT PRIMARY KEY,
            country TEXT,
            macro_genre TEXT
        )
    ''')
    conn.commit()
    conn.close()
    logger.info("✅ Tabla 'artist' lista")


def artist_in_database(artist: str) -> Tuple[bool, Optional[str], Optional[str]]:
    """Verifica si el artista existe y devuelve (existe, país, macro_género)."""
    conn = sqlite3.connect(str(ARTIST_DB_PATH))
    cursor = conn.cursor()
    cursor.execute('SELECT country, macro_genre FROM artist WHERE name = ?', (artist,))
    res = cursor.fetchone()
    conn.close()
    if res:
        return True, res[0], res[1]
    return False, None, None


def insert_artist(artist: str, country: str, genre: Optional[str] = None, source: str = "") -> None:
    """Inserta nuevo artista o actualiza solo campos faltantes."""
    conn = sqlite3.connect(str(ARTIST_DB_PATH))
    cursor = conn.cursor()
    cursor.execute('SELECT country, macro_genre FROM artist WHERE name = ?', (artist,))
    existing = cursor.fetchone()
    
    if existing:
        existing_country, existing_genre = existing
        update_country = country != 'Unknown' and (not existing_country or existing_country == 'Unknown')
        update_genre = genre and genre != 'Unknown' and (not existing_genre or existing_genre == 'Unknown')
        
        if update_country or update_genre:
            set_clauses = []
            params = []
            if update_country:
                set_clauses.append("country = ?")
                params.append(country)
            if update_genre:
                set_clauses.append("macro_genre = ?")
                params.append(genre)
            params.append(artist)
            cursor.execute(f"UPDATE artist SET {', '.join(set_clauses)} WHERE name = ?", params)
            conn.commit()
    else:
        cursor.execute("INSERT INTO artist (name, country, macro_genre) VALUES (?, ?, ?)", 
                      (artist, country, genre))
        conn.commit()
    
    conn.close()


def get_latest_chart_database() -> Optional[Path]:
    """Encuentra la base de datos de charts de YouTube más reciente por identificador de semana."""
    if not CHARTS_DB_DIR.exists():
        return None
    
    db_files = list(CHARTS_DB_DIR.glob("youtube_charts_*.db"))
    if not db_files:
        return None
    
    def week_key(file_path: Path) -> tuple:
        match = re.search(r'youtube_charts_(\d{4})-W(\d{1,2})\.db', file_path.name)
        if match:
            return (int(match.group(1)), int(match.group(2)))
        return (0, file_path.stat().st_mtime)
    
    latest = max(db_files, key=week_key)
    logger.info(f"📁 Usando base de datos: {latest.name}")
    return latest


def get_artists_from_chart_db(db_path: Path) -> Set[str]:
    """Extrae todos los nombres de artistas únicos de la base de datos de charts."""
    artists = set()
    conn = sqlite3.connect(str(db_path))
    cursor = conn.cursor()
    
    # Encontrar la columna de artistas (puede tener diferentes nombres)
    cursor.execute("PRAGMA table_info(chart_data)")
    columns = [col[1] for col in cursor.fetchall()]
    artist_column = None
    for col in ['Artist Names', 'Artist', 'Artists', 'artist', 'Artista']:
        if col in columns:
            artist_column = col
            break
    
    if artist_column:
        safe_column = f'"{artist_column}"' if ' ' in artist_column else artist_column
        cursor.execute(f"SELECT {safe_column} FROM chart_data WHERE {safe_column} IS NOT NULL")
        for row in cursor.fetchall():
            if row[0]:
                artists.update(split_artists(str(row[0])))
    
    conn.close()
    logger.info(f"📊 {len(artists)} artistas únicos encontrados")
    return artists
```

- **Responsabilidad única** (solo operaciones de base de datos)

- **Reutilizable** en toda la aplicación

- **Fácil de probar** (se puede simular)

- **Manejo de errores centralizado** para operaciones de base de datos

  

#### 3.3 Analizador de Artistas (`utils/artist_parser.py`)

**Propósito**: Dividir cadenas que contienen múltiples artistas.

```python
"""
Utilidades de análisis de nombres de artistas.
"""

import re
from typing import List


def split_artists(artist_str: str) -> List[str]:
    """
    Divide una cadena que contiene múltiples artistas en nombres individuales.
    
    Maneja: "feat.", "ft.", "&", "and", "y", comas, paréntesis.
    
    Ejemplos:
        "ROSÉ & Bruno Mars" → ["ROSÉ", "Bruno Mars"]
        "Bad Bunny feat. J Balvin" → ["Bad Bunny", "J Balvin"]
        "ROSÉ, Bruno Mars, Lady Gaga" → ["ROSÉ", "Bruno Mars", "Lady Gaga"]
    """
    if not artist_str or not isinstance(artist_str, str):
        return []
    
    # Normalizar patrones de colaboración
    artist_str = re.sub(r'\s+feat\.?\s+', ', ', artist_str, flags=re.IGNORECASE)
    artist_str = re.sub(r'\s+ft\.?\s+', ', ', artist_str, flags=re.IGNORECASE)
    artist_str = re.sub(r'\s+featuring\s+', ', ', artist_str, flags=re.IGNORECASE)
    
    # Reemplazar conjunciones
    artist_str = re.sub(r'\s+&\s+', ', ', artist_str)
    artist_str = re.sub(r'\s+y\s+', ', ', artist_str)
    artist_str = re.sub(r'\s+and\s+', ', ', artist_str, flags=re.IGNORECASE)
    
    # Eliminar paréntesis (a menudo contienen información de colaboración)
    artist_str = re.sub(r'[\(\)]', '', artist_str)
    
    # Dividir por coma y limpiar
    artists = [a.strip() for a in artist_str.split(',')]
    artists = [a for a in artists if a and len(a) > 1]
    
    # Saltar marcadores de posición
    skip_words = {'various', 'various artists', 'unknown', 'varios', 'varios artistas'}
    artists = [a for a in artists if a.lower() not in skip_words]
    
    return artists
```

```python
"""
Cliente de API de MusicBrainz para búsqueda de país y género.
"""

import time
from difflib import SequenceMatcher
from typing import Optional, Tuple, List

from ..config import logger, get_cache, get_http_sessions
from ..utils.text_utils import generate_all_variations
from ..country_detector import validate_and_normalize_country


def search_musicbrainz_country_cached(artist: str) -> Tuple[Optional[str], str]:
    """Busca en MusicBrainz el país de origen del artista."""
    cache = get_cache()
    if artist in cache['musicbrainz_country']:
        return cache['musicbrainz_country'][artist]
    
    sessions = get_http_sessions()
    session = sessions['musicbrainz']
    
    try:
        url = "https://musicbrainz.org/ws/2/artist/"
        headers = {'User-Agent': 'ArtistDB/5.0 (contact@example.com)'}
        params = {'query': artist, 'fmt': 'json', 'limit': 5}
        resp = session.get(url, headers=headers, params=params, timeout=10)
        
        if resp.status_code != 200:
            return None, "No encontrado"
        
        data = resp.json()
        if not data.get('artists'):
            return None, "No encontrado"
        
        # Encontrar la mejor coincidencia usando coincidencia difusa
        artist_lower = artist.lower()
        best_score = 0.0
        best_artist = None
        
        for mb_artist in data['artists'][:5]:
            mb_name = mb_artist.get('name', '').lower()
            bonus = 0.3 if (artist_lower in mb_name or mb_name in artist_lower) else 0.0
            score = SequenceMatcher(None, artist_lower, mb_name).ratio() + bonus
            if score > best_score:
                best_score = score
                best_artist = mb_artist
        
        if best_score < 0.55:
            return None, "No encontrado"
        
        # Extraer país de los campos area o begin-area
        for field in ['area', 'begin-area']:
            if field in best_artist:
                country_raw = best_artist[field].get('name', '')
                if country_raw:
                    country_norm = validate_and_normalize_country(country_raw)
                    if country_norm:
                        result = (country_norm, f"MusicBrainz ({artist})")
                        cache['musicbrainz_country'][artist] = result
                        return result
        
        return None, "No encontrado"
        
    except Exception as e:
        logger.debug(f"Error de MusicBrainz para {artist}: {e}")
        return None, "No encontrado"


def search_musicbrainz_genre_cached(artist: str) -> List[Tuple[str, int, str]]:
    """Busca en MusicBrainz las etiquetas de género asociadas con un artista."""
    cache = get_cache()
    if artist in cache['musicbrainz_genre']:
        return cache['musicbrainz_genre'][artist]
    
    sessions = get_http_sessions()
    session = sessions['musicbrainz']
    candidates = []
    
    try:
        url = "https://musicbrainz.org/ws/2/artist/"
        headers = {'User-Agent': 'ArtistDB/5.0'}
        params = {'query': artist, 'fmt': 'json', 'limit': 1}
        resp = session.get(url, headers=headers, params=params, timeout=10)
        
        if resp.status_code == 200:
            data = resp.json()
            if data.get('artists'):
                mb_data = data['artists'][0]
                
                # Extraer del campo 'genres' (estructurado)
                if 'genres' in mb_data:
                    for genre in mb_data['genres'][:5]:
                        name = genre.get('name', '')
                        count = genre.get('count', 1)
                        if name:
                            candidates.append((name.lower(), count, 'musicbrainz_genres'))
                
                # Fallback al campo 'tags' si no hay géneros
                if 'tags' in mb_data and not candidates:
                    for tag in mb_data['tags'][:5]:
                        name = tag.get('name', '')
                        count = tag.get('count', 1)
                        if name:
                            candidates.append((name.lower(), count, 'musicbrainz_tags'))
    
    except Exception as e:
        logger.debug(f"Error de género de MusicBrainz para {artist}: {e}")
    
    cache['musicbrainz_genre'][artist] = candidates
    return candidates
```

**Características clave:**

- **Coincidencia difusa** con SequenceMatcher para variaciones de nombre

- **Caché** para evitar consultas repetidas al mismo artista

- **Pool de conexiones** a través de sesiones HTTP compartidas

- **Degradación elegante** (devuelve vacío en error)

```python
Cliente de API de Wikipedia para extracción de país y género.

def search_wikipedia_infobox_country_cached(artist: str, lang: str = 'en') -> Optional[str]:
    """Extrae el país de los campos del infobox de Wikipedia."""
    cache = get_cache()
    key = (artist, lang)
    if key in cache['wikipedia_country']:
        return cache['wikipedia_country'][key]
    
    sessions = get_http_sessions()
    session = sessions['wikipedia']
    
    try:
        # Primero, resolver redirecciones y obtener el título real de la página
        url = f"https://{lang}.wikipedia.org/w/api.php"
        params = {'action': 'query', 'titles': artist, 'redirects': 1, 'format': 'json'}
        resp = session.get(url, params=params, timeout=8)
        data = resp.json()
        
        pages = data['query']['pages']
        page_id = next(iter(pages))
        title = pages[page_id].get('title', artist)
        
        # Obtener el wikitexto de la página
        params = {'action': 'parse', 'page': title, 'prop': 'wikitext', 'format': 'json', 'redirects': True}
        resp = session.get(url, params=params, timeout=10)
        data = resp.json()
        
        if 'parse' not in data:
            return None
        
        wikitext = data['parse']['wikitext']['*']
        
        # Extraer infobox (artista musical o banda)
        infobox_pattern = r'\{\{\s*Infobox (?:musical artist|band)[\s\S]*?\}\}'
        infobox_match = re.search(infobox_pattern, wikitext, re.IGNORECASE)
        if not infobox_match:
            return None
        
        infobox = infobox_match.group()
        
        # Buscar campos relacionados con país
        fields = ['origin', 'birth_place', 'location', 'from', 'country']
        for field in fields:
            field_pattern = r'\|?\s*' + field + r'\s*=\s*([^\n|]+)'
            field_match = re.search(field_pattern, infobox, re.IGNORECASE)
            if field_match:
                value = field_match.group(1).strip()
                # Eliminar marcado wiki
                value = re.sub(r'\[\[(?:[^|\]]*\|)?([^\]]+)\]\]', r'\1', value)
                value = re.sub(r'<[^>]+>', '', value)
                country_norm = validate_and_normalize_country(value)
                if country_norm:
                    cache['wikipedia_country'][key] = country_norm
                    return country_norm
        
        return None
        
    except Exception as e:
        logger.debug(f"Error de infobox de Wikipedia para {artist} ({lang}): {e}")
        return None


def search_wikipedia_summary_country_cached(artist: str, lang: str = 'en') -> Optional[str]:
    """Extrae el país del resumen de la página de Wikipedia usando patrones regex."""

        if not extract:
            return None
```

```python
}}

    else:

Devuelve SOLO un objeto JSON válido:
{{
    "country": "país de origen",
    "genre": "género musical principal"
}}

Sé preciso. Si no estás seguro, pon null. Sin texto adicional.
```

#### 6.1 Detector de País (`country_detector.py`)

**Propósito**: Orquesta la búsqueda en cascada del país del artista.

```python
"""
Lógica de detección de país para artistas.
"""

import time
from typing import Optional, Tuple

from .config import logger
from .utils.text_utils import generate_all_variations, detect_script_from_name
from .apis.musicbrainz import search_musicbrainz_country_cached
from .apis.wikipedia import (
    search_wikipedia_summary_country_cached,
    search_wikipedia_infobox_country_cached,
)
from .apis.wikidata import search_wikidata_country_cached
from .apis.deepseek import search_deepseek_fallback


def search_country(artist: str) -> Tuple[Optional[str], str]:
    """
    Determina el país de origen usando fuentes en cascada.
    
    Orden de búsqueda:
    1. MusicBrainz (con variaciones de nombre)
    2. Wikipedia EN (resumen, luego infobox)
    3. Wikipedia idiomas prioritarios (basado en detección de escritura)
    4. Wikidata
    5. Fallback de IA DeepSeek
    """
    variations = generate_all_variations(artist)
    
    # 1. MusicBrainz
    for var in variations[:3]:
        country, source = search_musicbrainz_country_cached(var)
        if country:
            info = f" (var: {var})" if var != artist else ""
            return country, f"MusicBrainz{info}"
    time.sleep(0.8)
    
    # 2. Wikipedia Inglés
    for var in variations[:3]:
        country = search_wikipedia_summary_country_cached(var, 'en')
        if country:
            info = f" (var: {var})" if var != artist else ""
            return country, f"Wikipedia EN resumen{info}"
        
        country = search_wikipedia_infobox_country_cached(var, 'en')
        if country:
            info = f" (var: {var})" if var != artist else ""
            return country, f"Wikipedia EN infobox{info}"
    time.sleep(0.3)
    
    # 3. Wikipedia idiomas prioritarios (basado en detección de escritura)
    detected_lang = detect_script_from_name(artist)
    priority_langs = []
    if detected_lang:
        priority_langs.append(detected_lang)
    priority_langs.extend(['es', 'pt', 'fr', 'de', 'it', 'hi', 'ko', 'ja', 'zh', 'ar', 'tr', 'ru'])
    seen = set()
    priority_langs = [lang for lang in priority_langs if not (lang in seen or seen.add(lang))]
    
    for lang in priority_langs:
        for var in variations[:2]:
            country = search_wikipedia_summary_country_cached(var, lang)
            if country:
                info = f" (var: {var})" if var != artist else ""
                return country, f"Wikipedia {lang.upper()}{info}"
            time.sleep(0.2)
    
    # 4. Wikidata
    for var in variations[:3]:
        country = search_wikidata_country_cached(var)
        if country:
            info = f" (var: {var})" if var != artist else ""
            return country, f"Wikidata{info}"
    
    # 5. Fallback de DeepSeek
    logger.debug(f"  🔍 Usando fallback de DeepSeek para país: {artist}")
    deepseek_country, _, _ = search_deepseek_fallback(artist)
    if deepseek_country:
        return deepseek_country, "DeepSeek API"
    
    return None, "No encontrado"
```



#### 6.2 Detector de Género (`genre_detector.py`)

**Propósito**: Orquesta la búsqueda de género con sistema de votación ponderada.

```python
"""
Sistema de detección de género y votación para artistas.
"""

import time
from collections import defaultdict
from typing import Optional, Tuple, List

from .config import logger
from .utils.text_utils import generate_all_variations, detect_script_from_name
from .apis.musicbrainz import search_musicbrainz_genre_cached
from .apis.wikidata import search_wikidata_genre_cached
from .apis.wikipedia import (
    search_wikipedia_infobox_genre_cached,
    search_wikipedia_summary_genre_cached,
)
from .apis.deepseek import search_deepseek_fallback
from .dictionaries.genres import GENRE_MAPPINGS
from .dictionaries.macro_genres import MACRO_GENRES, GENERIC_MACROS
from .dictionaries.country_rules import COUNTRY_GENRE_PRIORITY, COUNTRY_SPECIFIC_RULES, DEFAULT_GENRE_PRIORITY
from .dictionaries.stopwords import GENRE_STOPWORDS


def normalize_genre(genre_text: str) -> Tuple[Optional[str], Optional[str]]:
    """Mapea una cadena de género en bruto a (macro_género, subgénero)."""
    if not genre_text:
        return None, None
    
    text_norm = genre_text.lower().strip()
    
    # Saltar palabras vacías
    if text_norm in GENRE_STOPWORDS:
        return None, None
    
    # Búsqueda directa
    if text_norm in GENRE_MAPPINGS:
        return GENRE_MAPPINGS[text_norm]
    
    # Coincidencia de subcadena para variantes más largas
    for genre_variant in sorted(GENRE_MAPPINGS.keys(), key=len, reverse=True):
        if genre_variant in text_norm and len(genre_variant) > 3:
            return GENRE_MAPPINGS[genre_variant]
    
    return None, None


def select_primary_genre(artist: str, genre_candidates: List[Tuple[str, int, str]],
                         country: Optional[str] = None, detected_lang: Optional[str] = None) -> Optional[str]:
    """Selecciona el macro-género principal usando sistema de votación ponderada."""
    if not genre_candidates:
        return None
    
    macro_votes = defaultdict(float)
    
    for subgenre, weight, source in genre_candidates:
        macro, _ = normalize_genre(subgenre)
        if macro:
            final_weight = weight
            
            # Bonificaciones basadas en fuente
            if 'musicbrainz' in source:
                final_weight *= 1.5
            if 'infobox' in source:
                final_weight *= 1.2
            if 'wikidata' in source:
                final_weight *= 1.3
            
            # Bonificaciones por término para géneros específicos
            sub_lower = subgenre.lower()
            if any(term in sub_lower for term in ['reggaeton', 'reggaetón', 'trap latino']):
                final_weight *= 1.4
                macro = 'Reggaetón/Trap Latino'
            if any(term in sub_lower for term in ['k-pop', 'kpop', 'korean pop']):
                final_weight *= 1.4
                macro = 'K-Pop/K-Rock'
            if any(term in sub_lower for term in ['sertanejo', 'funk brasileiro']):
                final_weight *= 1.4
            
            macro_votes[macro] += final_weight
    
    # Aplicar bonificaciones de prioridad por país
    if country and macro_votes:
        priority = COUNTRY_GENRE_PRIORITY.get(country, DEFAULT_GENRE_PRIORITY)
        for macro in list(macro_votes.keys()):
            if macro in priority:
                idx = priority.index(macro)
                if idx == 0:
                    macro_votes[macro] *= 2.0
                elif idx == 1:
                    macro_votes[macro] *= 1.5
                else:
                    macro_votes[macro] *= 1.2
    
    # Aplicar reglas específicas por país
    if country and macro_votes and country in COUNTRY_SPECIFIC_RULES:
        rule = COUNTRY_SPECIFIC_RULES[country]
        if "force_macro" in rule:
            for macro in list(macro_votes.keys()):
                if macro != rule["force_macro"]:
                    peso = macro_votes.pop(macro, 0)
                    macro_votes[rule["force_macro"]] += peso * rule.get("bonus_extra", 1.0)
    
    if macro_votes:
        return max(macro_votes.items(), key=lambda x: x[1])[0]
    
    return None


def search_artist_genre(artist: str, country: Optional[str] = None) -> Tuple[Optional[str], str]:
    """Determina el género principal usando fuentes en cascada."""
    all_candidates = []
    variations = generate_all_variations(artist)
    detected_lang = detect_script_from_name(artist)
    
    # 1. MusicBrainz
    for var in variations[:2]:
        candidates = search_musicbrainz_genre_cached(var)
        all_candidates.extend(candidates)
        if candidates:
            break
        time.sleep(0.5)
    
    # 2. Wikidata
    if not all_candidates:
        for var in variations[:2]:
            candidates = search_wikidata_genre_cached(var)
            all_candidates.extend(candidates)
            if candidates:
                break
            time.sleep(0.5)
    
    MIN_CANDIDATES = 3
    
    # 3. Wikipedia en idiomas prioritarios
    priority_langs = []
    if country:
        # Mapear país a códigos de idioma
        country_lang_map = {
            'South Korea': ['ko', 'en'],
            'Japan': ['ja', 'en'],
            'China': ['zh', 'en'],
            'Brazil': ['pt', 'en'],
            'Mexico': ['es', 'en'],
            'France': ['fr', 'en'],
            'Germany': ['de', 'en'],
            # ... más mapeos
        }
        priority_langs.extend(country_lang_map.get(country, ['en', 'es']))
    else:
        if detected_lang:
            priority_langs.append(detected_lang)
        priority_langs.append('en')
    
    priority_langs = list(dict.fromkeys(priority_langs))  # Eliminar duplicados
    
    if len(all_candidates) < MIN_CANDIDATES:
        for var in variations[:1]:
            for lang in priority_langs:
                candidates = search_wikipedia_infobox_genre_cached(var, lang)
                all_candidates.extend(candidates)
                if len(all_candidates) >= MIN_CANDIDATES:
                    break
                if not candidates:
                    candidates = search_wikipedia_summary_genre_cached(var, lang)
                    all_candidates.extend(candidates)
                if len(all_candidates) >= MIN_CANDIDATES:
                    break
                time.sleep(0.3)
            if len(all_candidates) >= MIN_CANDIDATES:
                break
    
    # 4. Fallback de DeepSeek
    if not all_candidates:
        logger.debug(f"  🔍 Usando fallback de DeepSeek para género: {artist}")
        _, deepseek_genre, _ = search_deepseek_fallback(artist, context_country=country)
        if deepseek_genre:
            macro, _ = normalize_genre(deepseek_genre)
            if macro:
                return macro, f"DeepSeek API (género: {deepseek_genre})"
            else:
                return deepseek_genre, f"DeepSeek API (crudo: {deepseek_genre})"
    
    # 5. Fallback por país
    if not all_candidates and country:
        priority = COUNTRY_GENRE_PRIORITY.get(country, DEFAULT_GENRE_PRIORITY)
        if priority:
            return priority[0], f"Género: fallback por país ({country})"
    
    if all_candidates:
        primary_genre = select_primary_genre(artist, all_candidates, country, detected_lang)
        sources = set(s for _, _, s in all_candidates)
        return primary_genre, f"Género: {', '.join(sources)}"
    
    return None, "Género no encontrado"
```

## 🐛 Solución de Problemas

### Problemas Comunes y Soluciones

| Error                                                    | Causa Probable                                    | Solución                                                     |
| :------------------------------------------------------- | :------------------------------------------------ | :----------------------------------------------------------- |
| `No chart databases found`                               | El Script 1 no se ha ejecutado todavía            | Ejecutar `1_download.py` primero                             |
| `ModuleNotFoundError: No module named 'build_artist_db'` | Ruta de ejecución incorrecta                      | Ejecutar desde la raíz del proyecto: `python scripts/2_1.build_artist_db.py` |
| `DeepSeek API key not set`                               | Falta la clave de API                             | Configurar variable de entorno `DEEPSEEK_API_KEY` (opcional, el script continúa) |
| `MusicBrainz rate limiting`                              | Demasiadas solicitudes                            | El script incluye demoras; sea paciente o aumente `RETRY_DELAY` |
| `Wikipedia page not found`                               | Nombre de artista incorrecto o página inexistente | Verificar nombre del artista; el script recurre a otras fuentes |
| `API timeout`                                            | Red lenta o problemas de API                      | Aumentar `DEFAULT_TIMEOUT` en config                         |
| `Database locked`                                        | Acceso concurrente                                | Esperar y reintentar; el script usa transacciones cortas     |
| `Artist not found in any source`                         | Artista muy oscuro                                | Agregar reglas de fallback o usar DeepSeek (si la clave está configurada) |

## 🧪 Limitaciones Conocidas y Mejoras Futuras

### Limitaciones Actuales

- **Dependencia de API**: El sistema depende de servicios externos que pueden cambiar o limitar la tasa
- **Artistas Nuevos**: Los artistas emergentes recientemente pueden no aparecer en las bases de conocimiento
- **Géneros de Nicho**: Algunos micro-géneros pueden no tener mapeos todavía
- **Detección de Escritura**: Basada en heurísticas, puede identificar incorrectamente ocasionalmente
- **Costo de DeepSeek**: Aunque mínimo (~$0.002 por 100 artistas), requiere clave de API
- **Límites de Tasa de Wikipedia**: Puede ser lento para procesamiento por lotes
- **Resolución de Colaboraciones**: Colaboraciones complejas (5+ artistas) pueden no resolverse con precisión

### Casos Extremos Conocidos

| Caso                                          | Comportamiento Actual           | Mejora Propuesta                         |
| :-------------------------------------------- | :------------------------------ | :--------------------------------------- |
| Grupos de K-Pop con miembros extranjeros      | Asignado a Corea del Sur        | Detectar países de miembros individuales |
| MCs brasileños                                | Puede recurrir a Sertanejo      | Priorizar Funk Brasileiro                |
| Colaboraciones latinas con líder de otro país | Puede dar Multi-country         | Analizar orden de mención                |
| Artistas con múltiples nacionalidades         | Gana el primer país detectado   | Ponderar por prominencia/actividad       |
| Géneros regionales oscuros                    | Mapeado a macro-género genérico | Agregar mapeos más específicos           |

### Mejoras Futuras

- Agregar API de Spotify como fuente adicional
- Implementar puntuación de confianza para todas las detecciones
- Agregar modo de procesamiento por lotes para catálogos grandes
- Mejorar la detección de escritura con modelos de ML
- Agregar persistencia para caché entre ejecuciones
- Soporte para más idiomas de Wikipedia
- Agregar geocodificación para nombres de ciudades sin contexto de país
